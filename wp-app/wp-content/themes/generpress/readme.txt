	== Description ==
	GenerPress is a Highly Ads optimized Wordpress Theme (Adsense and other Ads). It is designed to increase the click-thru-rate(CTR) and give you the ability to display ads in many location flexibility. With Clean design and very flexible theme option panel, Super Ads is your ultimate Theme to help to make more money from online advertising.

	* Responsive Layout
	* Custom Colors
	* Custom Header
	* Post Formats
	* The GPL v2.0 or later license. :) Use it to make something cool.

	== Frequently Asked Questions ==

	Q: How to install this theme?
	A: To install the WordPress Theme in your WordPress, you have two options. You may either upload the theme ZIP file using the WordPress theme installer (Recommended), or you can upload the unzipped theme folder via FTP (Advanced).


	== Upgrade Notice ==

	In your WordPress dashboard, go to Appearance => Themes. you will see the updates available will display an alert banner. Click on the theme you want to upgrade and click on "update now".
